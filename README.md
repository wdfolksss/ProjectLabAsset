# 🧪 Lab Asset Management System

Sistem Informasi Inventaris Laboratorium berbasis **Python Flask** yang digunakan untuk mengelola data aset laboratorium secara mudah dan efisien. Aplikasi ini memiliki dua jenis pengguna, yaitu **Admin** dan **Asisten**, dengan hak akses yang berbeda.

---

## 🛠️ Teknologi yang Digunakan

- Python 3
- Flask
- PyMySQL
- MySQL / phpMyAdmin
- HTML5
- Tailwind CSS
- Font Awesome

---

## 🚀 Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/username/lab-asset.git
```

### 2. Masuk ke Folder

```bash
cd lab-asset
```

### 3. Buat Virtual Environment

```bash
python -m venv venv
```

### 4. Aktifkan Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Mac/Linux

```bash
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Import Database

- Buka phpMyAdmin
- Buat database **lab_asset**
- Import file SQL

### 7. Jalankan Project

```bash
python app.py
```

Buka browser

```
http://127.0.0.1:5000
```
